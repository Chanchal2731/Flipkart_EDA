# Flipkart_EDA

A data-driven analysis of Flipkart product data to uncover insights related to
pricing strategy, customer ratings, reviews, and product categories.

## Objective
To identify patterns and trends in product pricing and customer behavior
using Exploratory Data Analysis techniques.

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## Key Highlights
- Cleaned and preprocessed real-world e-commerce data
- Analyzed price distribution and rating trends
- Visualized customer engagement through reviews and ratings
- Generated actionable insights for business decision-making

## Outcome
Delivered meaningful insights that can support pricing optimization and
product performance analysis in e-commerce platforms.
